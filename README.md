# 🚀 OneResume (마스터 이력서 관리 플랫폼)

> **한 번의 입력으로 모든 기업에 대응하는 스마트 이력서 관리 서비스**

## 👥 팀원 소개
- **팀장 20211106(박정웅)**: 데이터베이스/AWS 서버 관리
- **팀원 권민석**: 백앤드 담당
- **팀원 정현우**: Figma디자인 레이아웃/ 설계
- **팀원 최현민**: 프론트앤드 담당

## 🛠 기술 스택 (Tech Stack)
- **Frontend**: React.js, Tailwind CSS
- **Backend**: Node.js
- **Database**: MySQL
- **Infrastructure**: AWS (EC2, S3, R53)
- **DevOps**: Nginx
- **AI**: OpenAI GPT-4o-mini
- **Tools**: GitHub, Figma, VS Code, Notion, PuTTY, WinSCP

## ✨ 주요 기능
1. **Master Resume**: 이력서 데이터 통합 관리
2. **Real-time Preview**: 입력과 동시에 확인하는 이력서 미리보기
3. **AI Analysis**: 채용 공고(JD) 분석 및 맞춤 키워드 제안
4. **PDF Export**: 다양한 템플릿의 PDF 다운로드
